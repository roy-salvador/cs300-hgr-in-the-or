# Network Training
All network training efforts were performed in a Google Cloud Instance with 24 GB of memory and P100 GPU. Check `requirements.txt` for the listing of needed python libraries and their versions used.

## Datasets
We trained and evaluated with the following datasets:
* [Sheffield Kinect Gesture (SKIG) Dataset](https://pgram.com/dataset/shefeld-kinect-gesture-skig-dataset/)
* [Our Dynamic Hand Gesture Dataset](https://www.kaggle.com/dataset/dfdab0aa99299796ae9465fe43299f2e21308f61d1f72740cc476644860a7b67)
* [TwentyBN Jester Dataset](https://20bn.com/datasets/jester)
